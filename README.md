# Audacious website

Clean opgesplitste websitebestanden.

## Structuur

- `html/index.html` — de HTML-pagina
- `css/style.css` — alle styling
- `js/main.js` — alle JavaScript / interactie

## Kleurthema

Deze versie gebruikt een lichte Audacious-stijl met witte basisvlakken en oranje accenten voor CTA's, lijnen, icons, hover states en technische visuals.

Open `html/index.html` in je browser om de website lokaal te bekijken.
